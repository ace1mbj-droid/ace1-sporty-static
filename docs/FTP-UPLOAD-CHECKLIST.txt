## Need Help?

If something doesn't work after upload:

1. Check browser console (F12) for errors
2. Verify js/supabase-config.js uploaded correctly
3. Check file permissions
4. Make sure .htaccess or web.config is in root directory
5. Update Supabase redirect URLs to your new domain
6. Clear browser cache and try again
