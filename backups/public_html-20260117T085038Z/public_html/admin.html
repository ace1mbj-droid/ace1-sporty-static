Image Guide

Product Images: Used on product cards, product detail pages, and quick view modal. Upload and manage these in the Products tab.

Hero Banner: Large banner image on the homepage (coming soon)

Category Images: Images for product categories (coming soon)

About/Team Images: Images for the About page team section (coming soon)

Logo: Site logo (currently text-based, no image needed)

Current Image Storage

All site images are stored in Supabase Storage bucket: Images

Image Type Location/Purpose Dimensions Usage
Product Images Product Detail Pages, Cards, Quick View Modal 500x500px (recommended) Upload in Products tab → Edit Product → Upload Images
Hero Banner Homepage top section 1920x600px Available in settings (coming soon)
Category Icons Category filters on Products page 200x200px Available in settings (coming soon)
Team Photos About page team section 300x400px Available in settings (coming soon)
Testimonial Photos Testimonials section on homepage 150x150px (circular) Available in settings (coming soon)

Quick Tips

  • Always optimize images before uploading (use TinyPNG or similar)
  • Use descriptive filenames for easier management
  • Product images should be square (1:1 aspect ratio) for best display
  • Keep file sizes under 500KB for faster loading
  • All images are automatically compressed by Supabase Storage
